# Material Docente: Prácticas Interactivas Big Data

## 1. Objetivo

- Proporcionar a los alumnos un conjunto de prácticas para introducirlos en el procesado de grandes cantidades de datos (Big Data)

## 2. Roadmap

Para realizar la práctica el alumno deberá tener instalado en su ordenador:
- Herramienta GIT para gestión de repositorios [Github](https://git-scm.com/downloads)
- Apache NiFi [NiFi](https://nifi.apache.org)
- Máquina virtual con sistema operativo linux y distribución Ubuntu 22.04 (Disponible en el enlace compartido en moodle)
